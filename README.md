# OsCreator
Дарова эта прога для создания ОС без программирования

# Интерфейс
Чтобы создать файл нажимаемм верху Плюс или правой кнопки мыши на белой области Создать файл
Потом выберете тип файла например .xml для десайна(роботает не везде) или .efi для загружика efi(для новый компов) или .legacy для старых компов
