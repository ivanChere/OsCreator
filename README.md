# OsCreator
Дарова эта прога для сосдания ОС без прогромирования

# Интерфейс
Чтобы сосдать файл нажемаем верху Плюс или правой кнопки мыши на белой облости Сосдать файл
Потом выберете тип файла напремер .xml для десайна(роботает не везде) или .efi для загружика efi(для новый компов) или .legacy для старых компов
